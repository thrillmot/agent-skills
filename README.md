# logmind-skill

skills.sh-compatible agent skill for [logmind](https://github.com/thrillmot/logmind).

Teaches AI agents (Claude Code, Cursor, Codex, Cline, Continue, ...) when and how to log decisions in any project that has logmind installed.

## Install

```bash
npx skills add -g thrillmot/logmind-skill
```

The skill activates whenever an agent works in a project with `.logmind/config.yml` or an `AGENTS.md`/`CLAUDE.md` mentioning logmind.

## What's in this repo

Just `SKILL.md`. That's the skill.

For the actual logmind package and source, see [thrillmot/logmind](https://github.com/thrillmot/logmind).
